# Fictieve demo-data voor cluster HR en Financien

> **Alle bestanden in deze map zijn verzonnen.** Persoonsnamen, plaatsnamen, bedragen, mailinhoud en beleidsfragmenten zijn opgesteld voor de MAF AI-inspiratiedag op 20 mei 2026 en weerspiegelen geen echte MAF-medewerkers, sollicitanten, thuisfrontteams of personeelsregelingen. Overeenkomsten met werkelijke personen of situaties berusten op toeval.

## Wat staat hier

- `mail-sollicitatie-fictief.md`, `mail-thuisfront-fictief.md`, `mail-veld-fictief.md` — drie fictieve binnenkomende mails als typische HR-cases
- `maf-handboek-fragment.md` — fictief fragment van een personeelshandboek (hoofdstuk thuiswerken) als kennis-bron
- `hr-toon.md` — toonrichtlijn voor de mail-reactie agent
- `functieprofiel-fictief.md` — fictief functieprofiel voor de onboarding-pakket agent
- `maandcijfers-fictief.csv` — fictieve maandcijfers voor de rapport-helper agent
