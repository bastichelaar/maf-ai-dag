# Fictieve demo-data voor cluster Relatiebeheer

> **Alle bestanden in deze map zijn verzonnen.** Persoonsnamen, organisatie-namen, plaatsnamen, bedragen, gespreksinhoud en relatie-historie zijn opgesteld voor de MAF AI-inspiratiedag op 20 mei 2026 en bevatten geen echte donateurs- of relatie-data van Stichting MAF Nederland. Overeenkomsten met werkelijke personen of situaties berusten op toeval.

## Wat staat hier

- `crm-extract-fictief.csv` — fictief CRM-extract van 19 relaties (donateurs, kerken, partner-organisaties, fondsen) met categorisering en korte notitie per relatie
- `gespreksverslag-1.md`, `gespreksverslag-2.md` — fictieve verslagen van een donateurs-bezoek en een telefonisch contact, opgesteld zoals een relatiebeheerder zou kunnen noteren
- `historische-contacten/` — drie fictieve contact-notities die de agent als context kan gebruiken voor de aandachts-radar
