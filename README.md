# AI-inspiratiedag MAF Nederland

Voorbereiding voor een AI-inspiratiedag bij Stichting MAF Nederland (Mission Aviation Fellowship), georganiseerd door Bas en Rik. Doel: teams meenemen van Copilot-chat naar werkende workflows en gekoppelde tools, met een organische brug naar een betaald vervolgtraject.

**Datum:** 20 mei  
**Organisatoren:** Bas Tichelaar & Rik  
**Contactpersoon MAF:** Gilbert

---

## Dagstructuur

Vier sessies van 90 minuten, geen plenaire opener of afsluiter. Per sessie: opener, mining-vragen, live demo, workflow-canvas, quick win en brug naar vervolg.

Voor de definitieve flow zie [`sessies-draaiboek.md`](sessies-draaiboek.md).

## Positionering

Copilot is het beginpunt, niet het eindpunt. We tonen wat er voorbij Copilot Chat mogelijk is met Claude-agents, MCP en gekoppelde workflows, zonder Copilot af te branden.

Kernboodschap: Copilot en Claude zijn complementair. Copilot is de handige collega in de bekende M365-tools. Claude is de specialist voor moeilijker werk. Workflow-tools zijn de lijm. MCP is hoe je een agent bouwt die zelfstandig door de hele keten heen werkt.
