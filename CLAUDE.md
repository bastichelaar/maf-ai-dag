# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Voorbereiding voor een **AI-inspiratiedag bij Stichting MAF Nederland** (Mission Aviation Fellowship), georganiseerd door Bas en zijn zwager. Doel: teams inspireren, van Copilot-chat naar workflows en gekoppelde tools — en organisch doorgroeien naar een betaald vervolgtraject.

## Context

- **MAF gebruikt:** Microsoft Copilot (M365), al een AI-primer gehad, vermoedelijk niet verder dan chat
- **Onze kracht:** Claude Code, MCP, agents — minder ervaring met Copilot/M365 specifiek
- **Teams:** donateurs, marketing, finance, HR, digital innovation, relatiebeheer, evenementen

## Dagstructuur (voorstel aan MAF)

4 clusters van 90 min + plenair begin/einde:
1. Communicatie & relaties (donateurs + marketing + relatiebeheer)
2. Bedrijfsvoering (HR + finance)
3. Evenementen
4. Digital Innovation — technisch dieper, ambassadeur-rol

Per cluster: intake → live demo → workflow-canvas → quick win committeren → brug naar vervolg.

## Vervolg-prioriteiten

Focus vervolggesprekken op: **Digital Innovation** (ambassadeur intern), **Marketing** (content-pipeline), **Donateurs** (data + persoonlijke communicatie schaalbaar).

Drie pakketten achter de hand:
- **A — Quick Wins Sprint (4 wk):** 1 use case per cluster werkend opgeleverd
- **B — Workflow Implementation (8–12 wk):** end-to-end cross-team workflow
- **C — AI Center of Excellence (doorlopend):** Digital Innovation coachen als intern ambassadeur

## Positionering

Brug bouwen: Copilot = beginpunt, niet eindpunt. Laten zien wat voorbij Copilot Chat mogelijk is met Claude-agents, MCP, en gekoppelde workflows.

## Pre-werk vóór de dag

- 1–2 dagen Copilot inlezen: Copilot Studio (agent met SharePoint-bron), Power Automate met AI-stap, Custom GPTs
- Demo-templates bouwen per cluster (invulbaar live)
- Één werkende Claude+MCP-agent op MAF-achtige data (voor opener + cluster 4)
- Intake-mail uitsturen per team (1 week vooraf)

## Deliverables in deze map

Bestanden die hier aangemaakt worden:
- `intake-mail.md` — template intake-mail per team (3 vragen)
- `dagprogramma.md` — volledig dagprogramma met tijden
- `vragen-per-cluster.md` — vragenlijst per cluster voor de sessies
- `demo-templates/` — demo-scripts per cluster
- `vervolgpakketten.md` — one-pager met 3 vervolgpakketten (A/B/C)
